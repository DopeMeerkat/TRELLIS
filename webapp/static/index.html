About this demo

Built for rapid local iteration with the TRELLIS API.

This UI keeps the blue-on-navy palette but drops platform branding. It is a lightweight FastAPI frontend that:

  • Uploads an image to your configured TRELLIS endpoint.
  • Polls job status locally and streams the resulting OBJ.
  • Hosts a small gallery sourced from data/gallery.

Customize the TRELLIS endpoint via the TRELLIS_API_URL environment variable before running the server.
